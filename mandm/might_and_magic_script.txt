Combine with my items/spells/monster list(soon to be a FAQ) for best 
results :)

Copyright 2000 by Andrew Schultz schultza@earthlink.net but the original 
ideas/disk format, etc. is copyright New World Computing/3DO.

Please do not copy this without permission.  This took *considerable* 
time and thought on my part, and if you would like to use it for your 
site, please address me *by name* in a friendly letter and I will almost 
certainly say yes.  Text is extracted from images of Apple disks off 
Apple.Asimov.net.  It is in ALL CAPS because that is the way MM1 handles 
it.  This document contains no instructions on how to solve the game;  
however, you can probably solve the game from the clues given by the 
script if you have knowledge of the critical puzzles.

Version 0.5.0 info:

I've added text maps.  They are still somewhat inaccurate, and you can 
only do so much with text graphics, but if you really want to see maps, 
look at:

http://www.geocities.com/SoHo/Exhibit/2762/mm1/maps.htm which has maps 
that correspond to the text maps here and will (as of 10/27/2000) be a 
little more descriptive.  In this case text maps can display some things 
rather clearly, and it was my first real attempt at the genre.

Some of the formatting may still seem post-modernist, but I'm working on 
it, I swear :).  There is a lot to delete to make this FAQ succinct!

**************************************************

There is a basic code for all the annotations on the map.  First,
* = VERY important
- = lose hit points(pit, drain)
T = tree
! = encounter
$ = treasure
& = encounter with fixed monsters
v = stairs down
^ = stairs up
: = portal
N = non-fight meeting with NPC's
1 = sign
2 = business
@ = important landmark
. = "ordinary"
? = where you are given a choice
# = raise attributes(?)
| = teleport

Some of these are awkward, but this was my first try, and I should fix 
them soon.
